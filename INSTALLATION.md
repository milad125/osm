# راهنمای نصب و راه‌اندازی

## پیش‌نیازها

1. **وردپرس**: نسخه 5.8 یا بالاتر
2. **PHP**: نسخه 7.4 یا بالاتر
3. **ووکامرس**: نسخه 5.0 یا بالاتر
4. **کلید API Google Maps**: برای نمایش نقشه

## مراحل نصب

### 1. نصب افزونه

1. فایل افزونه را در پوشه `wp-content/plugins/` آپلود کنید
2. یا از طریق پنل مدیریت وردپرس، افزونه را آپلود و نصب کنید

### 2. فعال‌سازی

1. به بخش "افزونه‌ها" در پنل مدیریت بروید
2. افزونه "سیستم پیشرفته حمل و نقل ووکامرس" را فعال کنید
3. مطمئن شوید که افزونه ووکامرس نیز فعال است

### 3. تنظیمات اولیه

#### الف) تنظیمات عمومی

1. به بخش "حمل و نقل" > "تنظیمات" بروید
2. کلید API نقشه گوگل را وارد کنید:
   - به [Google Cloud Console](https://console.cloud.google.com/) بروید
   - یک پروژه جدید ایجاد کنید یا پروژه موجود را انتخاب کنید
   - API Maps JavaScript را فعال کنید
   - یک کلید API ایجاد کنید
   - کلید را در تنظیمات افزونه وارد کنید

3. مختصات مرکز نقشه را تنظیم کنید (به صورت پیش‌فرض تهران است)

#### ب) تنظیمات API سرویس‌های ارسال

برای هر سرویس ارسالی که می‌خواهید استفاده کنید:

**پست تاپین:**
1. به [پنل تاپین](https://tapin.ir) بروید
2. کلید API خود را دریافت کنید
3. در تنظیمات افزونه، بخش "API ها" > "پست تاپین" را فعال کنید
4. کلید API را وارد کنید

**الوپیک:**
1. به [پنل الوپیک](https://alopeyk.com) بروید
2. کلید API خود را دریافت کنید
3. در تنظیمات افزونه، بخش "API ها" > "الوپیک" را فعال کنید
4. کلید API را وارد کنید

**اسنپ باکس:**
1. به [پنل اسنپ باکس](https://snappbox.ir) بروید
2. کلید API خود را دریافت کنید
3. در تنظیمات افزونه، بخش "API ها" > "اسنپ باکس" را فعال کنید
4. کلید API را وارد کنید

**تیپاکس:**
1. به [پنل تیپاکس](https://tipax.ir) بروید
2. کلید API خود را دریافت کنید
3. در تنظیمات افزونه، بخش "API ها" > "تیپاکس" را فعال کنید
4. کلید API را وارد کنید

#### ج) افزودن مراکز ارسال

1. به بخش "حمل و نقل" > "مراکز ارسال" بروید
2. روی "افزودن مرکز جدید" کلیک کنید
3. اطلاعات مرکز را وارد کنید:
   - نام مرکز
   - آدرس کامل
   - مختصات جغرافیایی (عرض و طول)
   - شهر و استان
   - تلفن و ایمیل (اختیاری)
4. مرکز را فعال کنید
5. ذخیره کنید

**نکته:** می‌توانید چندین مرکز ارسال اضافه کنید. سیستم به صورت خودکار نزدیک‌ترین مرکز را انتخاب می‌کند.

#### د) تنظیمات بسته‌بندی

1. به بخش "حمل و نقل" > "تنظیمات" > "بسته‌بندی" بروید
2. هزینه بسته‌بندی را تعیین کنید:
   - **ثابت**: یک مبلغ ثابت به ریال
   - **درصدی**: درصدی از مبلغ کل سفارش
3. ذخیره کنید

#### ه) تنظیمات برنامه وفاداری (اختیاری)

1. به بخش "حمل و نقل" > "تنظیمات" > "وفاداری" بروید
2. برنامه وفاداری را فعال کنید
3. امتیاز به ازای هر سفارش را تعیین کنید
4. امتیاز به ازای هر ریال را تعیین کنید
5. ذخیره کنید

#### و) تنظیمات اعلان‌ها (اختیاری)

1. به بخش "حمل و نقل" > "تنظیمات" > "اعلان‌ها" بروید
2. اعلان‌ها را فعال کنید
3. نوع اعلان‌ها را انتخاب کنید (ایمیل، SMS)
4. برای SMS، کلید API سرویس SMS را وارد کنید (مثلاً کاوه‌نگار)
5. ذخیره کنید

### 4. تنظیمات ووکامرس

1. به بخش "ووکامرس" > "تنظیمات" > "ارسال" بروید
2. روش‌های ارسال فعال شده را در مناطق ارسال خود اضافه کنید
3. برای هر روش ارسال، تنظیمات مورد نظر را انجام دهید

## استفاده

### برای کاربران

1. کاربر به صفحه تسویه حساب می‌رود
2. نقشه Google Maps نمایش داده می‌شود
3. کاربر موقعیت تحویل را روی نقشه انتخاب می‌کند
4. سیستم به صورت خودکار:
   - شهر را تشخیص می‌دهد
   - نزدیک‌ترین مرکز ارسال را پیدا می‌کند
   - فاصله را محاسبه می‌کند
   - هزینه ارسال را محاسبه می‌کند
5. کاربر روش ارسال مورد نظر را انتخاب می‌کند
6. هزینه نهایی (شامل بسته‌بندی) نمایش داده می‌شود

### برای مدیران

**مدیریت سفارش‌ها:**
- به بخش "حمل و نقل" > "سفارشات ارسال" بروید
- لیست تمام سفارش‌ها را مشاهده کنید
- وضعیت هر سفارش را تغییر دهید
- شماره رهگیری را مشاهده کنید

**رهگیری:**
- در صفحه سفارش، اطلاعات رهگیری نمایش داده می‌شود
- نقشه رهگیری برای سفارش‌های در حال ارسال نمایش داده می‌شود

**برنامه وفاداری:**
- به بخش "حمل و نقل" > "برنامه وفاداری" بروید
- کاربران برتر را مشاهده کنید
- تراکنش‌های امتیاز را بررسی کنید

**گزارش‌ها:**
- به بخش "حمل و نقل" > "گزارشات" بروید
- آمار کامل سفارش‌ها و درآمد را مشاهده کنید

## عیب‌یابی

### نقشه نمایش داده نمی‌شود
- مطمئن شوید کلید API Google Maps را وارد کرده‌اید
- بررسی کنید که API Maps JavaScript در Google Cloud Console فعال است
- محدودیت‌های کلید API را بررسی کنید

### هزینه ارسال محاسبه نمی‌شود
- مطمئن شوید حداقل یک مرکز ارسال فعال دارید
- بررسی کنید که کلید API سرویس ارسال را وارد کرده‌اید
- لاگ خطاهای PHP را بررسی کنید

### اعلان‌ها ارسال نمی‌شوند
- بررسی کنید که اعلان‌ها را فعال کرده‌اید
- برای SMS، کلید API سرویس SMS را بررسی کنید
- تنظیمات SMTP وردپرس را بررسی کنید

## پشتیبانی

برای پشتیبانی و گزارش باگ، لطفا با توسعه‌دهنده تماس بگیرید.
