# سیستم پیشرفته حمل و نقل ووکامرس (OSM1)

افزونه پیشرفته و حرفه‌ای برای مدیریت سیستم حمل و نقل در ووکامرس با قابلیت‌های مدرن و پیشرفته.

## ویژگی‌ها

### 🚚 روش‌های ارسال
- **پست تاپین**: اتصال کامل به API پست تاپین
- **الوپیک**: ارسال از طریق سرویس الوپیک
- **اسنپ باکس**: یکپارچه‌سازی با اسنپ باکس
- **تیپاکس**: اتصال به سیستم تیپاکس
- **ارسال فلش**: تحویل در همان روز با هزینه بیشتر

### 🗺️ نقشه زنده
- نمایش نقشه Google Maps در صفحه تسویه حساب
- انتخاب موقعیت دقیق تحویل توسط کاربر
- محاسبه خودکار هزینه بر اساس فاصله و شهر
- جستجوی آدرس با Google Places API

### 📦 مدیریت بسته‌بندی
- محاسبه هزینه بسته‌بندی (ثابت یا درصدی)
- نمایش هزینه بسته‌بندی در لیبل روش ارسال
- افزودن خودکار به هزینه نهایی

### 🏢 مدیریت مراکز ارسال
- مدیریت چندین مرکز ارسال
- محاسبه خودکار نزدیک‌ترین مرکز
- تعیین مختصات جغرافیایی برای هر مرکز

### 📍 رهگیری Real-Time
- رهگیری لحظه‌ای سفارش‌ها
- نمایش وضعیت به‌روز سفارش
- نقشه رهگیری در صفحه سفارش
- به‌روزرسانی خودکار وضعیت‌ها

### 🔔 سیستم نوتیفیکیشن
- ارسال ایمیل برای رویدادهای مهم
- ارسال SMS (قابل اتصال به سرویس‌های SMS)
- اعلان به مدیر سایت
- اعلان تغییر وضعیت سفارش

### 🎮 برنامه وفاداری و گیمیشن
- سیستم امتیازدهی برای هر سفارش
- سطوح مختلف کاربری (جدید، برنز، نقره، طلا، پلاتین)
- استفاده از امتیازها برای تخفیف
- نمایش امتیازها در پنل کاربر

### 🤖 هوش مصنوعی و پیش‌بینی
- پیش‌بینی زمان تحویل بر اساس روش ارسال و فاصله
- پیشنهاد بهترین روش ارسال
- محاسبه هوشمند هزینه

### 📊 گزارش‌گیری
- آمار کامل سفارش‌ها
- گزارش درآمد
- آمار بر اساس روش ارسال
- آمار ماهانه و سالانه

## نصب

1. فایل افزونه را در پوشه `wp-content/plugins/` آپلود کنید
2. افزونه را از پنل مدیریت وردپرس فعال کنید
3. به بخش "حمل و نقل" در منوی مدیریت بروید
4. تنظیمات را پیکربندی کنید:
   - کلید API نقشه گوگل را وارد کنید
   - کلیدهای API سرویس‌های ارسال را وارد کنید
   - مراکز ارسال را اضافه کنید

## نیازمندی‌ها

- وردپرس 5.8 یا بالاتر
- PHP 7.4 یا بالاتر
- ووکامرس 5.0 یا بالاتر
- کلید API Google Maps (برای نمایش نقشه)

## تنظیمات

### تنظیمات عمومی
- کلید API نقشه گوگل
- مرکز نقشه (عرض و طول جغرافیایی)

### تنظیمات API
برای هر سرویس ارسال باید کلید API مربوطه را وارد کنید:
- پست تاپین
- الوپیک
- اسنپ باکس
- تیپاکس

### هزینه بسته‌بندی
- می‌توانید هزینه بسته‌بندی را به صورت ثابت یا درصدی تنظیم کنید

### برنامه وفاداری
- فعال/غیرفعال کردن برنامه وفاداری
- تعیین امتیاز به ازای هر سفارش
- تعیین امتیاز به ازای هر ریال

## استفاده

### برای کاربران
1. در صفحه تسویه حساب، نقشه نمایش داده می‌شود
2. کاربر موقعیت تحویل را روی نقشه انتخاب می‌کند
3. هزینه ارسال به صورت خودکار محاسبه می‌شود
4. کاربر روش ارسال مورد نظر را انتخاب می‌کند

### برای مدیران
- مدیریت مراکز ارسال از بخش "مراکز ارسال"
- مشاهده و مدیریت سفارش‌ها از بخش "سفارشات ارسال"
- مدیریت برنامه وفاداری از بخش "برنامه وفاداری"
- مشاهده گزارش‌ها از بخش "گزارشات"
